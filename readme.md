# OM Game
A 2D game made for [**Daydream Game Jam**](https://daydream.hackclub.com/) where you have to sacrifice your FOV in order to complete the game. Playable via web on [**itch.io**](https://mejpek.itch.io/om-game), no need to download anything! 
## How to play
Player can move using WASD or →←↓↑ and attack the enemies with left mouse button. Goal of the game is to defeat the final boss, that loses HP based on the players level. Have fun!
